# ThaiBathConvert

โปรแกรมนี้ใช้สำหรับแปลงจำนวนเงินเป็นข้อความภาษาไทย (บาท/สตางค์)

## วิธีการรันโปรแกรม

1. ติดตั้ง Go (แนะนำเวอร์ชัน 1.18 ขึ้นไป)
2. ติดตั้ง dependency ด้วยคำสั่ง:
   ```
   go mod tidy
   ```
3. รันโปรแกรมด้วยคำสั่ง:
   ```
   go run main.go
   ```
